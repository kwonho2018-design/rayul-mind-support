## 자주 묻는 질문

### 음악이 재생되지 않아요.
앱에 포함된 오프라인 MP3 음원을 재생합니다. 기기의 무음 모드, 볼륨, 오디오 출력 장치, 잠금 화면 오디오 상태를 확인해 주세요.

### 음악 저작권은 어떻게 처리되어 있나요?
앱에 포함된 음원은 개발자가 상업적 배포 가능한 권리와 라이선스 범위 안에서 제공하는 콘텐츠입니다.

### 개인정보를 수집하나요?
카페에서 하루종일 듣고 싶은 음악은 로그인, 광고, 구독, 위치 정보, 마이크, HealthKit, 개인정보 수집을 사용하지 않습니다.

### 의료 앱인가요?
아니요. 카페에서 하루종일 듣고 싶은 음악은 진단, 치료, 의료 조언을 제공하지 않는 음악 재생 앱입니다.
