# RAYUL MIND 지원

RAYUL MIND는 휴식과 수면 전 루틴을 위한 오프라인 음악 재생 앱입니다.

## 지원 안내

앱 사용 중 문제가 있거나 문의가 필요한 경우 아래 이메일로 연락해 주세요.

- 이메일: kwonho2018@gmail.com
- 앱 이름: RAYUL MIND
- 플랫폼: iOS

문의 시 사용 중인 iPhone 모델, iOS 버전, 앱 버전, 발생한 문제를 함께 알려주시면 더 빠르게 확인할 수 있습니다.

## 자주 묻는 질문

### 음악이 재생되지 않아요.
앱에 포함된 오프라인 음원을 재생합니다. 기기의 무음 모드, 볼륨, 오디오 출력 장치를 확인해 주세요.

### 개인정보를 수집하나요?
RAYUL MIND는 로그인, 광고, 구독, 위치 정보, 마이크, HealthKit, 개인정보 수집을 사용하지 않습니다.

### 의료 앱인가요?
아니요. RAYUL MIND는 진단, 치료, 의료 조언을 제공하지 않는 음악 재생 앱입니다.
