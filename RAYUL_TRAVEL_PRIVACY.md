# RAYUL TRAVEL 개인정보 처리방침

최종 업데이트: 2026년 6월 12일

RAYUL TRAVEL은 해외여행 계획, 일정, 준비물, 문서, 경비, 실제 방문 기록을 사용자의 기기 안에서 관리하는 iPhone 앱입니다. 현재 버전은 계정, 광고, 분석 SDK, 외부 서버 동기화, 제3자 추적 기능을 사용하지 않습니다.

## 수집 및 저장

앱은 사용자가 직접 입력하거나 선택한 여행명, 일정, 준비물, 문서 메타데이터, 경비, 정산 정보, 실제 방문 기록을 앱의 로컬 저장소에 저장합니다.

사용자가 선택한 여행 사진, 영수증 이미지, 문서 파일, PDF 리포트는 앱 문서 폴더에 저장됩니다. 이 데이터는 외부 서버로 전송되지 않습니다.

## 권한 사용

위치 권한은 사용자가 체크인을 저장하거나 실제 방문 장소를 남길 때만 요청합니다.

사진 접근 권한은 사용자가 여행 사진, 영수증 이미지, 문서 스크린샷을 직접 첨부할 때만 사용합니다.

Face ID 또는 기기 인증은 사용자가 민감 문서로 표시한 항목을 열람할 때만 사용합니다. 생체 정보는 앱에 저장되지 않으며 Apple의 LocalAuthentication 시스템을 통해 처리됩니다.

## 외부 전송 및 추적

RAYUL TRAVEL은 현재 버전에서 사용자의 여행 데이터, 위치 기록, 사진, 영수증, 문서, 경비 정보를 개발자 서버나 제3자 서비스로 전송하지 않습니다.

앱은 광고 식별자, 추적 목적의 기기 식별자, 제3자 광고 네트워크, 외부 분석 SDK를 사용하지 않습니다.

## 데이터 삭제

사용자는 앱 설정의 모든 여행 데이터 삭제 기능으로 기기에 저장된 여행, 일정, 문서 파일, 사진, PDF 리포트, 경비, 기록을 삭제할 수 있습니다.

앱을 삭제하면 iOS 동작에 따라 기기 안의 로컬 앱 데이터도 삭제됩니다.
